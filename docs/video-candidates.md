# Кандидаты: видео-скиллы для автоматизации контента в соцсетях

> Разведка по GitHub от 2026-06-23. Цель — автоматизация Reels/Shorts (нарезка, монтаж, транскрипция).
> Это **черновик-шортлист**, НЕ карточки каталога. Перед записью в `entries/` — дочитать исходник построчно и запинить на коммит.

## ⚠️ Общее по безопасности
Все эти скиллы дают Claude право запускать `ffmpeg`/`yt-dlp`/`whisper` и ставить зависимости локально. Явного вредоноса (кража секретов, левые домены) не найдено ни у кого. Два риска: (1) обработка чужих скачанных видео — потенциальные эксплойты в метаданных; (2) облачная транскрипция (ElevenLabs/Groq/OpenAI) шлёт твою речь на сторонние серверы. Где исходник не дочитан построчно — помечено.

## 🔥 Авто-нарезка длинного видео → клипы (самое ценное для соцсетей)

### ⭐ clipify — РЕКОМЕНДАЦИЯ
- **Source:** https://github.com/louisedesadeleer/clipify
- Длинное видео → клипы 9:16: Whisper-транскрипт, сам ищет панчлайны/смешные моменты, режет, переформат 16:9→9:16 с трекингом говорящего, караоке-субтитры в стиле Opus.pro.
- Юзкейсы: автонарезка подкаста/стрима на Reels; вирусные клипы без ручного монтажа.
- ~433⭐, релиз май 2026, Python, MIT.
- 🟢 Безопасность: работает целиком локально, нет запроса секретов / левых доменов / prompt injection в доступном коде. Самый чистый в нише.

### claude-shorts — премиум, интерактивный
- **Source:** https://github.com/AgriciDaniel/claude-shorts
- 10-шаговый пайплайн: faster-whisper (GPU) → Claude скорит сегменты, ты выбираешь → субтитры в Remotion (Bold/Bounce/Clean, karaoke) → экспорт Shorts/TikTok/Reels.
- ~132⭐, v1.0.0 фев 2026, MIT.
- 🟡 Нужен GPU/CUDA, venv в `~/.shorts-skill/`, Google Fonts CDN + Claude API. Проверить `install.sh`/`setup.sh` перед запуском.

## 🎬 Монтаж/обработка (ffmpeg)

### ⭐ claude-ffmpeg-skill — РЕКОМЕНДАЦИЯ (самый чистый кирпич)
- **Source:** https://github.com/ychoi-kr/claude-ffmpeg-skill
- Чистая обёртка ffmpeg: конверт форматов, ресайз 16:9→9:16, GIF, аудио, обрезка/склейка, скорость, субтитры (вшить/мягкие/извлечь), кадры, сжатие. Пресеты под YouTube/Instagram/TikTok/Twitter.
- ~19⭐, v0.2 окт 2025 (небольшой, но ffmpeg-команды стабильны).
- 🟢 Безопасность: никаких сетевых вызовов и секретов, всё локально. MIT. Минус — без «мозгов» (не ищет моменты, не транскрибирует).

### video-use (browser-use) — «опиши словом»
- **Source:** https://github.com/browser-use/video-use
- Кладёшь видео + одно предложение → сам режет паузы/«ээ», цветокор, субтитры, оверлеи. Точность до слова (ElevenLabs Scribe).
- ~10.1k⭐, MIT, репутация команды хорошая.
- 🟡 Речь уходит на ElevenLabs (нужен их API-ключ).

### claude-videoedit — «комбайн», но необкатанный
- **Source:** https://github.com/opheliabm/claude-videoedit
- Полный suite: edit/transcode (H.264/265/AV1/GPU)/субтитры Whisper/анализ качества VMAF/генерация через Google Veo/shortform-пайплайн.
- ⚠️ 0⭐, совсем новый. Много облачных API (Pixabay/Veo/FLUX/SD) → ключи в env. Исходник построчно НЕ проверен. Осторожно.

## 📝 Транскрипция (видео/аудио → текст)

### ⭐ claude-video `/watch` — РЕКОМЕНДАЦИЯ
- **Source:** https://github.com/bradautomates/claude-video
- `/watch` скачивает видео (YouTube/TikTok/Vimeo/локально через yt-dlp), извлекает кадры, транскрибирует → отдаёт Claude «глаза и уши».
- ~2.4k⭐, v0.1.3 май 2026, MIT.
- 🟡 yt-dlp качает с произвольных URL; Whisper-ключи (Groq/OpenAI) в `~/.config/watch/.env`; теоретический prompt injection через содержимое видео. Есть флаг `--no-whisper`.

### transcript-critic — локально, без облака
- **Source:** https://github.com/jftuga/transcript-critic
- Транскрипция через **whisper.cpp локально** + структурный разбор: таймкоды, саммари, слабые места, цитаты.
- ~31⭐, v1.0.0 фев 2026.
- 🟢 whisper.cpp локально, нет hardcoded-секретов. 🟡 yt-dlp без валидации источника; транскрипт подставляется в промпт (теор. injection).

> Отдельного чистого популярного «whisper → SRT/VTT» скилла с заточкой под русский — НЕ найдено. Транскрипция везде вшита внутрь видео-комбайнов.

## Вывод — что брать под задачу (русский контент + нарезка)
1. **clipify** — основной инструмент автонарезки 9:16 с караоке-субтитрами, самый безопасный (всё локально). Лучший старт.
2. **claude-ffmpeg-skill** — безопасный «кирпич» под свои ffmpeg-операции без облака.
3. **claude-video `/watch`** или **transcript-critic** — транскрипция чужих/своих роликов для контент-плана.

## 🕳 Пустые ниши = шанс сделать свой скилл (через skill-creator)
- Нет чистого «whisper → SRT/VTT» под **русский** (кириллица, переносы, русские стоп-слова).
- Нет интеграции «нарезка → постинг в VK/Telegram/Reels» — все останавливаются на экспорте файла.
- ⚠️ Русский: локальный Whisper хорошо берёт RU только на `medium`/`large`; мелкие модели врут с таймкодами субтитров.
